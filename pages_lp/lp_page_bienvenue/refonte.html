Comment bénéficier De l'offre de bienvenue ?

Etape 1 Créez votre compte

Saisissez vos coordonnées personnelles (email, téléphone, nom, prénom). Pour un compte PRO, munissez-vous de votre numéro de SIRET, obligatoire pour valider votre statut de professionnel.

Etape 2 Choisissez vos produits

Ajoutez les articles de votre choix dans votre panier en cliquant sur les produits souhaités. Ensuite, accédez à votre panier en cliquant sur l'icône en haut à droite de votre page.

Etape 3 Appliquez le code promo

Saisissez le code BIENVENUE10 dans le champ "Code promo," puis cliquez sur "Appliquer". Votre code promo sera pris en compte et vous verrez la réduction dans votre panier.

Etape 4 Validez votre panier

Après avoir appliqué le code promo, cliquez sur "Valider votre panier" pour finaliser votre commande. Assurez-vous de suivre les étapes de paiement et de livraison pour compléter l'achat.
